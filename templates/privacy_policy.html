Política de Privacidade

WaChat - Assistente WhatsApp Automatizado

Ver Termos de Serviço

1. Sobre o WaChat

O WaChat é uma prova de conceito (POC - Proof of Concept) que tem como objetivo demonstrar a capacidade de receber e responder mensagens via WhatsApp de forma automatizada.

Importante: Este é um projeto experimental e de demonstração técnica.

2. Coleta e Uso de Dados

Os dados coletados e processados pelo WaChat incluem:

  • Mensagens recebidas: O conteúdo das mensagens enviadas pelos usuários via WhatsApp
  • Identificadores: Número de telefone do remetente e identificadores de sessão
  • Metadados: Data, hora e tipo de mensagem (texto, áudio, imagem)
Finalidade: Os dados são utilizados exclusivamente para o processamento e geração de respostas automatizadas às mensagens recebidas.

3. Compartilhamento de Dados

Não compartilhamos seus dados com terceiros. Todas as informações coletadas são processadas internamente e utilizadas apenas para a finalidade específica de gerar respostas automatizadas.

Não vendemos, alugamos ou comercializamos dados pessoais de usuários.

4. Armazenamento e Segurança

Os dados são armazenados temporariamente durante o processamento das mensagens. Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados contra:

  • Acesso não autorizado
  • Alteração indevida
  • Divulgação não consentida
  • Perda ou destruição acidental

5. Seus Direitos

De acordo com as leis de proteção de dados aplicáveis (incluindo a LGPD - Lei Geral de Proteção de Dados), você tem direito a:

  • Solicitar acesso aos seus dados pessoais
  • Solicitar a correção de dados incorretos ou incompletos
  • Solicitar a exclusão de seus dados
  • Revogar o consentimento para processamento de dados
  • Solicitar a portabilidade dos seus dados

6. Serviços de Terceiros

O WaChat utiliza os seguintes serviços de terceiros:

  • WhatsApp Business API (Meta): Para comunicação via WhatsApp
  • Heroku: Para hospedagem da aplicação
  • Serviços de IA: Para processamento de linguagem natural e geração de respostas

Esses serviços possuem suas próprias políticas de privacidade, às quais também estamos sujeitos.

7. Retenção de Dados

Os dados são mantidos apenas pelo tempo necessário para cumprir a finalidade para a qual foram coletados. Dados de sessão e mensagens podem ser armazenados temporariamente para fins de análise e melhoria do serviço.

8. Cookies e Tecnologias Similares

O WaChat não utiliza cookies para rastreamento de usuários. Utilizamos apenas tecnologias essenciais para o funcionamento básico da aplicação web.

9. Alterações nesta Política

Reservamo-nos o direito de atualizar esta Política de Privacidade periodicamente. Recomendamos que você revise esta página regularmente para se manter informado sobre como protegemos seus dados.
